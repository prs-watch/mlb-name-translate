# MLB Name Translate

Translate MLB player's English name into Japanese name.

```python
import mlbname

mlbname.translate("Noah Syndergaard")
# --> ノア・シンダーガード

mlbname.translate("Max Verlander")
# --> マックス・バーランダー

mlbname.translate("Max AAA")
# --> マックス・AAA

mlbname.translate("AAA BBB")
# --> AAA・BBB
```
